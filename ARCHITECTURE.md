# Структура проекта

```
Assets/
├── _Project/           <-- Ваша основная папка. Подчеркивание поднимает её наверх списка.
│   ├── Art/            <-- Зона ответственности Макара
│   │   ├── Materials/  
│   │   ├── Models/     
│   │   └── Textures/   
│   ├── Audio/          <-- Зона ответственности Алана
│   │   ├── Ambient/    (Дождь, гул вентиляции)
│   │   ├── SFX/        (Щелчки, кипение чайника)
│   │   └── UI/         (Пиканье радара, звонок телефона)
│   ├── Prefabs/        <-- Готовые "строительные блоки" игры
│   │   ├── Environment/
│   │   ├── Interactables/ (Собранный префаб чайника или выключателя)
│   │   └── UI/         
│   ├── Scenes/         
│   │   ├── Levels/     (Главная сцена диспетчерской)
│   │   └── Test_Scenes/<-- Песочницы для разработчиков
│   ├── Scripts/        <-- Ваш с Рузи и Мишей фронт работ
│   │   ├── Core/       (Шина событий, Менеджер состояний игры)
│   │   ├── Mechanics/  (Скрипты радара, игрока, взаимодействия)
│   │   ├── UI/         (Логика интерфейсов связи, рации)
│   │   └── Utils/      (Вспомогательные скрипты, таймеры)
│   └── Settings/       <-- Настройки проекта (Input Actions, Render Pipelines)
├── Plugins/            <-- Сторонние библиотеки (например, если будете юзать DOTween)
└── ThirdParty/         <-- Ассеты, скачанные из Unity Asset Store
```
